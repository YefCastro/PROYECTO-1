Test del Sistema de Rutas

Algoritmo: Dijkstra

Proceso: 1. El jugador selecciona edificio de origen 2. El jugador selecciona edificio de destino 3. El sistema genera matriz del mapa: 0 = celda no transitable (edificios, terreno vacío) 1 = celda transitable (vías) 4. Se envía petición POST al backend: /api/calculate-route 5. El backend retorna array de coordenadas representando la ruta 6. El frontend anima la ruta en el mapa
